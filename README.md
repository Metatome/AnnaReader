# AnnaReader - 您的终极纯净电子书阅读器

## 📖 项目缘起

在这个信息爆炸的时代，书籍和知识依然是人类最宝贵的共同财富。偶然间，我发现了 **Anna's Archive** —— 世界上最大的开源影子图书馆，那里汇聚了海量的书籍资源。我深感如此珍贵的资源应该让更多人便捷地获取和阅读。

然而，在寻找阅读器的过程中，我却屡屡受挫。市面上的阅读器要么广告漫天飞，严重干扰阅读体验；要么强制要求注册、充值会员，设立重重门槛。这些商业化的操作，让原本纯粹的读书这件事变得不再纯粹，也极其影响读书人的心情。

本着希望大家**“多读书，读好书”**的宗旨，我开发了 **AnnaReader**。这是一个完全免费、开源、无广告、注重隐私的现代化电子书阅读器，旨在为您提供最纯净、最专注的阅读体验。

## ✨ 核心特色

### 1. 全能格式支持
AnnaReader 支持市面上绝大多数主流电子书格式，让您无需为格式转换而烦恼：
- **PDF**
- **EPUB**
- **MOBI** (自动转换)
- **AZW3 / AZW**
- **TXT**
- **Markdown (MD)**

### 2. 深度集成 Anna's Archive
这是 AnnaReader 最独特的功能之一。您可以直接在应用内：
- **搜索**：无缝连接 Anna's Archive 的海量书库。
- **下载**：支持直接下载书籍到本地书架（通过 Slow Download 通道）。
- **查看详情**：获取书籍的详细元数据、封面和简介。

### 3. 跨平台支持
无论您使用什么设备，AnnaReader 都能陪伴在您身边：
- **Windows**
- **macOS**
- **Linux**
- **移动端支持**（Android/iOS 开发中）

### 4. 极致的安全与隐私保护
我们深知隐私的重要性，因此 AnnaReader 采用了**“本地优先”**的设计理念：
- **无追踪**：没有任何用户行为追踪或数据收集。
- **本地存储**：您的书架、阅读进度、笔记全部存储在本地。
- **安全网络请求**：仅在您主动搜索/下载书籍或同步时才会发起网络请求。支持代理设置，保障您的网络安全。

### 5. 强大的书架管理
- **智能导入**：支持文件和文件夹批量导入。
- **书架分类**：自定义书架，轻松管理成百上千本书籍。
- **全文搜索**：快速查找本地书籍。

### 6. 舒适的阅读体验
- **自定义设置**：字体、字号、背景色、行间距均可调节。
- **夜间模式**：保护您的视力。
- **目录跳转**：快速定位章节。
