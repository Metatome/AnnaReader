# AnnaReader 用户使用手册

## 1. 安装与启动

### Windows
1. 从发布页下载 `AnnaReader-Windows-x64.zip`。
2. 解压文件到任意目录。
3. 双击 `AnnaReader.Desktop.exe` 即可运行。

### macOS
1. 下载 `AnnaReader-macOS-x64.dmg`。
2. 打开 DMG 文件并将 AnnaReader 拖入 Applications 文件夹。
3. 在启动台点击图标运行。

### Linux
1. 下载 `AnnaReader-Linux-x64.tar.gz`。
2. 解压后运行 `./AnnaReader.Desktop`。
3. 确保已安装必要的依赖（如 GTK3）。

---

## 2. 图书管理

### 导入书籍
- **导入文件**：点击主界面的“导入文件”按钮，选择单个或多个电子书文件。
- **导入文件夹**：点击“导入文件夹”，选择包含电子书的目录，系统会自动扫描并导入支持格式的书籍。
- **拖拽导入**：直接将文件拖入应用窗口即可。

### 书架管理
- **新建书架**：点击侧边栏的“+”号，输入书架名称。
- **移动书籍**：右键点击书籍封面，选择“移动到...”，选择目标书架。
- **删除书籍**：右键点击书籍，选择“删除”。您可以选择仅从书架移除或同时删除本地文件。

### 搜索
- 在顶部搜索栏输入书名或作者名，即可实时筛选本地书库。

## 3. 阅读功能

### 阅读器界面
- **翻页**：点击页面左右两侧，或使用键盘左右方向键。
- **目录**：点击左上角的目录图标，跳转到指定章节。
- **全屏**：按 `F11` 进入/退出全屏模式。

### 个性化设置
点击阅读器右上角的“设置”图标（齿轮状），可以调整：
- **字体大小**：拖动滑块调整字号。
- **背景颜色**：选择护眼羊皮纸、夜间模式或纯白背景。
- **字体样式**：切换衬线体/无衬线体。

## 4. Anna's Archive 在线搜索与下载

### 搜索书籍
1. 点击侧边栏的“在线书库”或“Anna's Archive”。
2. 在搜索框输入关键词（书名、作者、ISBN）。
3. 系统会显示来自 Anna's Archive 的搜索结果。

### 下载书籍
1. 在搜索结果中点击书籍封面查看详情。
2. 点击“下载”按钮。
3. 系统将通过 Slow Download 通道下载书籍，下载完成后会自动添加到您的默认书架。
4. **注意**：下载速度取决于您的网络连接和 Anna's Archive 的服务器负载。

### 网络设置
如果您的网络无法直接访问 Anna's Archive，请在“设置” -> “网络”中配置代理：
- 勾选“使用代理”。
- 输入代理服务器地址和端口（例如 `127.0.0.1:7890`）。

---

## 5. 云同步 (WebDAV)

AnnaReader 支持通过 WebDAV 协议同步您的阅读进度和书架。

1. 打开“设置” -> “同步”。
2. 勾选“启用同步”。
3. 输入您的 WebDAV 服务器信息：
   - **URL**: WebDAV 地址 (例如 `https://dav.jianguoyun.com/dav/`)
   - **用户名**: 您的 WebDAV 账号
   - **密码**: 您的 WebDAV 密码/应用授权码
4. 点击“测试连接”。成功后，系统将在后台自动同步您的数据。

## 6. 常见问题 (FAQ)

**Q: 为什么搜索不到某些书籍？**
A: Anna's Archive 的收录虽然丰富但并非全能。此外，请检查您的网络连接是否通畅。

**Q: 支持哪些格式？**
A: 目前支持 PDF, EPUB, MOBI, AZW3, TXT, MD。MOBI 和 AZW3 会在打开时自动转换为 EPUB 以获得最佳阅读体验。

**Q: 软件是免费的吗？**
A: 是的，AnnaReader 永久免费且开源。
